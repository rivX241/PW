Aqui hablaré un poco acerca de los Estandares Web

La W3C es un organismo de categoría internacional que abarca muchos países que buscan estandarizar la creación de sitios web para que esta sea accesible y universal.

HTML

Es el lenguaje de marcado estándaar que se emplea para la creación de páginas web como este archivo. Se emplea la estructura y todo el contenido gracias a las etiqueetas y atributos.

CSS

Nos permite presentar un documento HTML,separando la presentación del contenido en sí, y nos ayuda a hacer más fáciles de mantener lás páginas.

HTTP

Es el protocolo para la transferencia en la web,y como se da la comunicación entre los clientes y los servidores.

JavaScript

Es el lenguaaje de programación que no se de be confundir con Java,este peermite crear contenido interactivo y dinámico las páginas para que no sean estáticas.

SVG

Este formato de gráficos vectoriales, que utiliza XML, permite la creación de imágenes que son escalables sin perder calidad. Es ampliamente utilizado en la web debido a su capacidad de ser manipulado con CSS y JavaScript.

WOFF

Formato diseñado para la web que optimiza la carga de fuentes, permitiendo tipografías personalizadas. Incluye metadatos que hacen más fácil su gestión.

WebRTC

Tecnología que permite establecer comunicación en tiempo real entre navegadores sin requerir complementos. Nos facilita hacer videoconferencias y mensajería instantánea directamente.

XML

Un lenguaje de marcado que permite la estructura y transporte de datos, siendo legible tanto por humanos como por computadoras.
